<h1>NF_EmotionTrackingVR</h1>

NF_EmotionTrackingVR is a Unity application that can recognize seven facial expressions (Happiness, Anger, Surprise, Sadness, Fear, Contempt, Disgust), using a VR visor with facial tracking sensors. After recognizing the chosen expression, a dictionary containing the pairs formed by (expression, associated numeric value) is saved in a json file
